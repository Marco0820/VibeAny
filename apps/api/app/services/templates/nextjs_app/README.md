# VibeAny Starter

This project was bootstrapped from Next.js 13.5.7 with Tailwind CSS.

## Available Scripts

- `npm run dev` – Runs the development server.
- `npm run build` – Builds the application for production.
- `npm run start` – Starts the production server.
- `npm run lint` – Lints the project using the Next.js ESLint configuration.

## Notes

Dependencies are not installed automatically. Run `npm install` inside the
project directory before starting the development server.
